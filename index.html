DeepTruth:

link to project: DeepTruth

Link to project code: https://github.com/TheChozenWon/DeepTruth

Overview:

DeepTruth is an innovative fact-checking platform that combines the power of Google's Gemini AI and DistilBERT to provide real-time analysis of news articles and claims. Our Chrome extension makes it easy to verify information while browsing the web.

Stack:

  • Frontend: React + Vite
  • Backend: Django + Django REST Framework
  • AI Models: Google Gemini Pro, Fine-tuned DistilBERT
  • Database: MongoDB
  • Browser Extension: Chrome Extension API

Preservio:

Link to project code: https://github.com/TheChozenWon/Preservio

Overview:

Preservio is a web application designed to help users a way to prevent food spoilage and waste.. It provides a user-friendly interface for adding food items with expiry dates and track when it would expire. Our track tackles sustainability and is prevalent in the real world because food waste is a real problem especially in food industry.

Stack:

  • Frontend: Typescript/CSS/JavaScript
  • Backend: Next.js/SQLite
  • Framework: React.js
